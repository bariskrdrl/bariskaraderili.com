Darken

Darken & Luminosity

background-blend-mode: darken, luminosity; background-image: url('images/pokemon-blend.jpg'), url('images/pikachu.png');

Burda gordugunuz gibi iki background resmi ve iki blend-mode ozelligi kullandim.Bu sekilde kullanimlarda ilk verdiginiz blend-mode ilk resme ikinciler ise kendi aralarinda eslesirler.

Boyle deneme yontemleriyle guzel seyler yakalanabilir!

hard-light

background-blend-mode: hard-light; background-image: url('images/gastly.png'), url('images/gastly.png'), url('images/gastly.png'), url('images/gastly.png'); background-position: 0% 0%, 10% 10%, 20% 20%, 30% 30%; animation: blend 3s infinite;

blend-mode & hover
.blend-hover .difference { opacity: 0; background-blend-mode: difference; } .blend-hover:hover .difference { opacity:1; } .blend-hover:hover .normal { opacity:0; }

Burdaki puf nokta hover oldugunda blend-mode herhangi bir transition gecisi almiyor.Iki versiyonuda absolute ile ust uste koyarak opacity ile oynuyoruz.
