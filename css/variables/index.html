Variables

Aniden degisen renkler, fontlar, font boyutlari, dinamizmi yakalamak icin yaptigin her turlu bok.

Neyin kafasini yasadigini bilmedigimiz musteri tarafindan aniden degistirilebilir.

Bu degisikliklerin zulme donusmemesi icin en guzeli bunlari bir degiskene almak ve tek yerden yonetmektir.

Suanda bunlari yapmamizi saglanyan sass, less, stylus gibi css derleyiciler kullaniyoruz.

w3 de artik bunu sana sundu.

Ornek

:root { --main-color: #000000; --title-color: #FFFFFF; --main-font: 'Tahoma'; --title-font: 'Ubuntu'; --main-size: 17px; --title-size: 26px; } h1, h2, h3, h4, h5, h6 { color: var(--title-color); font-size: var(--title-size); font-family: var(--title-font); }

Not: root: secicisi belgenin kok ogesini secer.Burada kok oge html dir.root: icerisine verdiginiz cssler direk html e isleyecektir.Bizde html de degiskenler olusturmus olduk.

Degiskenin degiskenini degistiremezsin. :)
