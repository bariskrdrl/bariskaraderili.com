Intersection Observer

Codropsda gezinirken karsilastigim bir javascript apisini denedim.

Oldukca kolay bir sekilde scroll animasyonlar yapilabiliyor.

Bunun icin suan da UIkit scrollspy componentini kullaniyorum ama single page bir projede gsap ile birlikte bunu denemek istiyorum.

Bir elemanin ekranda gorunur olduguna dair bize geri bildirim vererek calisiyor.

Suan da yazilan js icin tikla! 
                    const sections = document.querySelectorAll('.section');

                    const options = {
                        root: null,
                        threshold: .5,
                        //rootMargin: '100px'
                    };

                    const observer = new IntersectionObserver((entires, observer) => {
                        entires.forEach(entry => {
                            if (!entry.isIntersecting) {
                                return;
                            }

                            entry.target.classList.toggle('inverse');

                            observer.unobserve(entry.target);

                        });
                    }, options);

                    sections.forEach(section => {
                        observer.observe(section);    
                    });
