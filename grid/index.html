Suan da yazilan CSS icin tikla!
baba { display: grid; grid-template-columns: 200px 200px 200px 200px; grid-template-rows: 200px 200px 200px 200px; grid-column-gap: 20px; grid-row-gap: 20px; }

grid-template-columns: // yan yana kac cocuk gelecek ve genislikleri ne olacak?

grid-template-rows: // alt alta kac cocuk sirasi gelecek ve yuksekligi ne olacak?

grid-column-gap: // yan yana gelen cocuklarin arasinda ne kadar bosluk olacak?

grid-row-gap: // alt alta gelen cocuk siralarinin arasinda ne kadar bosluk olacak?

grid-gap: // cocuklar ve cocuk siralarinin arasinda ne kadar bosluk olacak?

grid-auto-columns: // belirtmedigin cocuk genislikleri ?

grid-auto-rows: // belirtmedigin satir yukseklikleri?

Bu sekilde olusturdugumuz grid yapisi responsive olmadigi icin fr adinda bir birim de eklemisler.Bu demek oluyorki bir cocuka 1fr dersek babasinin genisligini alir.

Az onceki ornekteki gibi yapacak olsakydik cocuklara 1fr 1fr 1fr 1fr verecektik.

Burada soyle dusunebiliriz baba 4 cocuklarin her biride 1 yani bir cocuk 1/4 genisligine sahip oluyor.

Bu senaryoda ekran tablete gectiginde grid-template-columns: 1fr 1fr; dersek artik baba 2 cocuklar 1 yani bir cocuk 1/2 olacak ve her zaman ekrani dolduracaklar.

auto fill auto fit

Oyle bir sey istedinki cocuklar 250px olsun babaya sigmayinca asagiya atsin ama asagi attiginda da diger cocuklarin toplami baba kadar olsun.

Ya da cocuklarin genisligi 50px ancak babayi dolduracak yeteri kadar cocuk yok ama tam sigdirmak isitiyorsun.Cocuklar sigmayinca duzgun sekilde asagi atmasini istiyorsun.

Baba 1000px cocuklar ise 250px aralarda da 20px gap lar oldugu icin 250px * 3 + 40px = 790px

1000px -790px = 220px! Ne oldu cocuk sigmadi asagi atti.Bu 220px e cocuk sigmadi diye bos mu kalsin?

Koda bakarsaniz cocuklarin kac kere tekrar edecegini yazmadim.Dedim ki auto-fill yani genislik degerine gore sen kendin ayarla.

Genislik olarakda minmax(250px, 1fr) dedim.Cocuklar babayi doldurmak icin 1fr oldu fazla gelincede fazla gelen asagi atti ve yukarida kalan cocuklar tekrar kendilerini babaya sigdirmak icin kendi genisliklerini baba / yukarida kalan cocuklar olarak guncelledi.

Suan da yazilan CSS icin tikla!
baba { display: grid; grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); grid-auto-row: 200px; grid-gap: 20px; }

grid position

  • grid-column-start // kacinci column dan basliyayim?
  • grid-column-end // kacinci column a kadar gideyim?
  • grid-row-start // kacinci row dan basliyayim?
  • grid-row-end // kacinci row a kadar gideyim?
  • grid-column // kacinci column a gideyim?
  • grid-row // kacinci row a gideyim?
