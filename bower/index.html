Bower - Web Paket Yoneticisi(Nedir Bu Kus?)

Kullandigimiz kutuphaneleri kolay bir sekilde yonetmemize yarayan bir paket yoneticisidir.

Boyle soylediklerinde kafamda hic bir sey canlanmamisti.Buna ne dersin?

Simdi gidip bootstrap indiriyorsunya sonra da jQuery indiriyosun sonra gidip bir tanede sekilli slider kutuphanesi sonra da bir tane fancybox sonra da da da da.. iste

Simdi bu kadar seyi indirdin gidip tek tek sitelerine ugradin aldin kendi projene klasor olusturup koydun bunlari.

Bower sayesinde boyle gidip tek tek bunlari indir sonra al projene ekleyle ugrasmayacan.

bower install bootstrap bu gidecek simdi bootstraopi indirecek sonrada diyecekki ya bootstrap jQuery kullanman lazim kardesim.

Ben sahibimi ugrastirmiyayim senin bagli oldugun kutuphaneyide sahibime hediye edeyim diyecek.

Simdi fancybox indireceksin yaz hemen bower install fancbox bakacak bizim kus diyecekki patron buda jQuery istiyor.

Kus akilli ben diyim.Ben gecen gun bootstrap indirirken jQuery indirdim tekrar indirmeme gerek yok sahibim onu kullanir.(Helal olsun lan sana mavis)

Nasil Kullanicam?

Oncelikle bower kullanmak icin nodejs kurmaliyiz.burdan indirebilirsin

Bower nodejs ile yazildigi icin nodejs paket yoneticlerine ihtiyacimiz var ve bu paket sayesinde bower i kuracagiz.Paket yonetcisini paket yoneticisinden indiriyoruz :)

nodejs kurdugunuzda otomatik olarak npm(nodejs package manager) kurulur.

Komut istemi, terminal artik adina ne dersen ac onu yaz oraya $ npm install -g bower global olarak bower kurulucak senin bilgisayarani.

Kuruldukdan sonra terminalden projeni olusturacagin dosya konumuna git.

bower search jquery asil enter a sana github uzerindeki adinda jquery yazan kutuphaneleri listeleyecek.

Ordan hangisini istiyorsan sec begen bower install jquery simdi jquery senin dosyana inmeyecek :D

Kus diyecekki patron ben bunu nasil indireyim git mit bir sey yok mu onu kur :)

Kur git ide cekinme sonrasinda mutlu olacaksin.

jQuery senin projene bower_components/ bu klasor icerisine gelecek.

Bu sekilde istedigin kadar kutuphaneyi hatta direk bir siteden url yazarak bir css,js dosyasini bile indirebilirsin.$ bower install http://example.com/script.js

Biraz degistirelim

Simdi en basa donduk nodejs kurdun.npm yaninda hediye olarak geldi.git ide gittin indirdin xD

$ npm install -g bower

bower init cikanlara bir seyler yazmak istersen yaz ben genelde enter yapiyom hepsine (:

$ bower install jquery --save

Simdi bower uzerinden indirdigin herseyi init yaptigin icin bower.json dosyasina kaydedecek.

Kaydedince ne olacak?

bower install dedigin anda json dosyasindaki kayitli her kutuphaneyi indirecek.Bir nevi kayit altina alacaksin herseyi.

Kardesim kusmusun nesin benim kutuphaneleri library diye bir klasore koy ben oyle bower_components/ bilmem diyebilirsin.

Usta yaparim paketci

            {
              "directory" : "library",
              "json"      : "bower.json"
